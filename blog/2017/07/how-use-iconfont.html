如何在自己的网页中使用阿里图标iconfont

日期：2017年07月04日

分类：使用笔记

一、打开iconfont官网并将图标加入项目

打开iconfont官网

在搜索框输入要查找的图标

将鼠标移至图标上方点击 ‘添加入库’

然后在网站右上方点击购物车

打开购物车看到添加的图标，选择添加至项目

新建一个项目并添加

二、引用样式并应用

然后就可以在‘我的图标’中看到‘我的项目’

点击生成代码

然后就可以看到生成的字体CDN代码，复制代码，粘贴在自己的css文件中

接着回到‘我的项目’中，鼠标移至图标上方选择复制代码

然后就会得到一个代码



在自己的css文件中设置iconfont类样式

.iconfont {
  font-family:"iconfont" !important;
  font-size:16px;
  font-style:normal;
  -webkit-font-smoothing: antialiased;
  -moz-osx-font-smoothing: grayscale;
}

使用<i>标签引用图标

<i class="iconfont"></i>

将刚才复制的图标代码替换进去

三、修改项目中图标

使用项目管理图标的好处在于我们可以随时方便的修改图标

我们可以修改大小，颜色，位置等信息

但是每次修改完必须要重新更换一次引用的CDN文件

点击更新代码后，再次引用即可

（完）
